Call group

Add users to the call group. Click Add User to select user from the list of all users. You'll be able to search for users that you want to add to this call group.
Change call sequence by moving users up and down.
If call parking is enabled, users are able to press *2(extension) to park the call and *4(extension) to retrieve the calls associated with this group.
You can upload the sound files to provide music on hold when the calls are parked.
