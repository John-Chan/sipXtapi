Accept REGISTER requests from (host)

Specifies the host for which the registry is willing to accept SIP REGISTER requests. You supply the fully qualified host name from the address of record for all SIP URIs that should register.

For example, if you want only lines that are set to Register with host to register with SipRegistrar, and your SIPxchange system creates all primary user lines and device lines in this format: sip:<resource ID>@sipxchange.acme.com, you would supply sipxchange.acme.com for this setting.

Property

SIP_REGISTRAR_DOMAIN_NAME

Type

String

Default

${SIPXCHANGE_DOMAIN_NAME}

(as defined in the Global Definitions file)
