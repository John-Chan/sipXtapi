HTTP port for embedded web server

With this setting, you enable access to a device’s embedded web server. Data transmission between the phone and a browser uses the HTTP protocol.

  • To enable the web server and use the HTTP protocol for data transmission, set this value to 80 (or any unused port number greater than 0) to identify the port number for HTTP messages.

  • To disable the web server, set this value to 0.

The embedded web server must be enabled before authorized phone users can access the browser-based interface for the phone.

Reminder: In a SIPxchange phone system, the phone's browser-based interface is used only for troubleshooting and other diagnostic procedures.

Parameter name

PHONESET_HTTP_PORT

Type

Integer

Default if blank

80 (for Pingtel xpressa phones)
0 (for Pingtel instant xpressa softphones)

Note

By default, the web server for instant xpressa softphones is disabled.

Device Category

Network
