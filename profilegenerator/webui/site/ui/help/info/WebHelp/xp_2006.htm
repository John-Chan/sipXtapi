Subscribe for voicemail NOTIFY msgs from

This setting allows a user's device(s) to discover whether there have been any status changes to a specified voicemail inbox. You define the address for the phone to send SUBSCRIBE messages for message waiting status indication for the user’s voicemail inbox: for SIPxchange voicemail, you supply your SIPxchange domain name.

Reminder: To set up SIPxchange voicemail for a user or group of users, you enable the User has voicemail inbox permission.

If an address is present, the phones associated with this user's assigned devices automatically send a SUBSCRIBE message to that address each time it starts up, and resubscribes before the subscription period expires. The voicemail server can then send the phone(s) a NOTIFY message each time the message waiting status changes.

Parameter name

PHONESET_MSG_WAITING_SUBSCRIBE

Type

String

Examples

sipxchange.acme.com

User Category

Network
