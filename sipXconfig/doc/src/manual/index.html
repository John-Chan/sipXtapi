sipXconfig started as a project to manage Pingtel's hard and soft phones and then later Cisco 7940 and 7960 phones. Manage here means generate configuration information native to the particular phone, e.g. provisioning, and deliver that information to the phone. In addition, notify the sipXproxy server with that phone's information for proper call handling.
