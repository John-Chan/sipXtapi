Backup files will be available for download once the backup is completed.

Backup files should be stored on media or a machine other than the sipX server.

Restoring the data:
In order to restore from backup you need access to a Linux console on your sipX host. Transfer the backup files back to the sipX host. Login as root and create two directories in root's home directory: backup-configs and backup-mailstore.

The files fs.tar.gz and pds.tar.gz go into the directory backup-configs. The file mailstore.tar.gz goes into the directory backup-mailstore.
To restore the configuration and database information type restore-configs.sh at a command line prompt.
To restore voicemail data type restore-mailstore.sh at a command line prompt.
