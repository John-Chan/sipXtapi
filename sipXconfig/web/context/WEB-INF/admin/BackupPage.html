To backup system configuration select backup options and press Backup Now button. You can perform partial backup by leaving some options unselected.
Backup files are available for download once backup is completed.

Click on each link to download all the files.
Backup files should be stored on media or a machine other than the sipX server.

In order to restore the data, transfer the backup files to the sipX server and put them in the relative folder, either backup-configs or backup-mailstore.
To restore the configuration and database information type restore-configs.sh at a command line prompt, restoring files from the fs.tar.gz and pds.tar.gz files.
To restore voicemail data type restore-mailstore.sh at a command line prompt, restoring files from the previous mailstore.tar.gz file.
