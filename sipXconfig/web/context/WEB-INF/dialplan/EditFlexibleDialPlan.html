Dial Plans

Dial plan consist of dial rules of various types. You can configure Dial Plan by adding, removing, editing or reordering the rules. It is possible to have more than one rule of each kind.

Rule order matters: make sure that more specific rules proceed more general rules. For example, move Long Distance rules for specified area codes above the default Long Distance rule.
