About application sets

Work with application sets

The Configuration Server supports the creation of application sets. Unlike user or device groups, application sets are not hierarchical, and applications can belong to any number of application sets.

For example, all of the users in the sales organization need the same four applications:

  • 411 directory

  • Unified messaging

  • CRM

  • Instant conferencing

By grouping all of these applications into one application set, and then assigning that application set to the user group that consists of all of the people in Sales, the Configuration Server speeds configuration time and automates software delivery: when you send application profiles to the Sales user group, all software applications in the application set are delivered to all phones assigned to members of the Sales user group.

With the Configuration Server, you can assign application sets or individual applications to each user or to a user group.
