Add a user group

To add a user group you specify identifying information:

  1. Click List All User Groups in the left column.
  2. Click Add User Group at the top of the User Groups page. A pop up window opens.
  3. Enter information about the group:
    • User Group Name
    • Parent User Group: select an existing group, or, to create a top-level group, select <BLANK>.
  4. Click Submit.

The pop up window remains open so that you can:

  • Add another user group.
  • Return to the List User Groups page: click Close.

To assign a user to a user group, you select a group when you add a user. You can also change a user's group assignment at any time.

Settings for the group

To make configuration choices for all of the users in a group:

  1. Click List All User Groups in the left column.
  2. Click on the name of the user group you want to configure. A new page opens with the group's General settings.
  3. To assure that phones of any type are correctly configured, indicate the settings you wish to configure first at the top of the page: select Pingtel or Cisco from the Settings for drop-down list.
    • Different categories of settings display based on your selection.
    • Settings for the General and Permission categories apply to the entire group, regardless of device type.
  4. To override values inherited from this group's parent groups), or to set values that apply to the users in this group only, click on a category name at the top of this page. The settings in that category display.
    Use the General category to change the identifying information for the group, or to change the group's hierarchical assignment.
  5. You define configuration settings by selecting one option from a provided list, making changes directly in a text box, or by clicking    to add or    to edit a set of fields in a pop up window.
  6. If a value has been defined for a configuration setting at a hierarchically higher level, for this group's parent user group or a user group above that,    displays next to the setting's name.

  7. For information on a specific user setting, click    next to the setting's name for help.

  8. To configure user settings that apply to a different type of phone, use the Settings for drop-down list to change from Pingtel to Cisco or vice versa.

  9. Click Save when your settings are complete.

Changes to a user group's settings do not take effect until you send profiles to the group.

Assign applications to a user group

Before you assign an application or application set to a user group, you must set up at least one application or set. To assign applications to a user group, you:

  1. Click List All User Groups in the left column.
  2. Find the group to which you want to assign applications.
  3. Select Assign Applications from the Actions list for that group.
  4. Click Go. A pop up window opens.
  5. To work with individual applications, click the Applications radio button at the top of the box.
    To work with application sets, click the Application Sets radio button.
  6. Choose an Application or Application Set to assign from the list that displays.
  7. Click Assign.

The pop up window remains open so that you can:

  • Assign other applications or application sets to the group.
  • Unassign applications or application sets.

  • Return to the User Groups page: click Close.

To change the application(s) assigned to a group: follow steps 1 through 4, then choose applications or application sets to assign or unassign.

Changes to a user group's application assignments do not take effect until you send profiles to the group.

Send profiles

To send configuration settings and device and application assignments to all of the phones assigned as devices to the users in a selected user group, you:

  1. Click List All User Groups in the left column.
  2. Find the user group.
  3. Select Send Profiles on the Actions list for that group.
  4. Click Go. A pop up window opens.
  5. Check the check box for:
    • Device settings if you have changed any device settings or assignments for users in the group, or changed the devices assigned to any users.
    • User settings if you have changed any user settings for this group.
    • Application settings if you have changed the applications assigned to this group, or to users in this group.

If you are not sure which settings changed, leave all three check boxes checked.

  1. Click Send Profiles.

The phones associated with all devices assigned to the users in the group restart, and the new settings take effect.

For reference, each time you send profiles the Job Status log records and displays profile status information.

Restart devices

To restart all of the phones assigned as devices to the users in a group, you:

  1. Click List All User Groups in the left column.
  2. Find the user group.
  3. Select Restart User Group Device(s) on the Actions list for that group.
  4. Click Go.

Other options

Delete a user group

Deleting a user group removes it from the system. Unless you reassign them first, you also delete:

  • All users assigned to the group

  • All subsidiary user groups

  • All users assigned to subsidiary user groups.

You may wish to change user group assignments first.

SIPxchange automatically unassigns any devices assigned to users in the deleted group(s) so that they can be assigned to other users.

To delete a user group, you:

  1. Click List All User Groups in the left column.
  2. Find the group you want to delete.

  3. Select Delete User Group on the Actions list for that group.

  4. Click Go.
