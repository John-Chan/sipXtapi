Work with device groups

To help you provision configuration settings and deliver software efficiently, you can create groups of devices and of users. A device or user can be a member of only one group at a time. However, groups can be set up in parent-child relationships to establish hierarchies.

Groups leverage commonalities among devices or users, allowing you to deliver configuration settings and application assignments to an entire group, rather than repetitively to individuals.

For example, all of the devices located in a certain building can be grouped together, or all of the users who are customer service representatives can be grouped together.

When you specify settings for a group, they automatically apply to each of the individual members assigned to the group. They also apply to each "child" group, and to the members of those child groups. This speeds the configuration task and makes selected features available consistently. For flexibility, however, you can override inherited settings at the level of any child group or for any individual member.
