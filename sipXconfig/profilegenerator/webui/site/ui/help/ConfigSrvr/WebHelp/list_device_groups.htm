Add a device group

To add a device group:

  1. Click List All Device Groups in the left column.
  2. Click Add Device Group at the top of the Device Groups page. A pop up window opens.
  3. Enter information about the device group, including:
    • Device Group Name
    • Parent Device Group: select an existing group, or, to create a top-level group, select <BLANK>.
  4. Click Submit.

The pop up window remains open so that you can:

  • Add another group.
  • Return to the List Device Groups page: click Close.

To assign a device to a device group, you select a group when you add a device. You can also change a device's group assignment at any time.

Settings for the group

To make configuration choices for all of the devices in a group:

  1. Click List All Device Groups in the left column.
  2. Click on the name of the device group you want to configure.
    A new page opens with the group's General settings.

  3. Device groups can include a mixture of devices with different device types. Indicate the settings you wish to configure first at the top of the page: select Pingtel or Cisco from the Settings for drop-down list.

    • Different categories of settings display based on your selection.

    • Settings for the General category apply to the group as a whole.

  4. To override values inherited from this group's parent group(s), or to set values that apply to the devices in this group only, click on a category name at the top of this page. The settings in that category display.
    Use the General category to change the identifying information for a group, or to change the group's hierarchical assignment.

  5. You define configuration settings by selecting one option from a provided list, making changes directly in a text box, or by clicking    to add or    to edit a set of fields in a pop up window.
  6. If a value has been defined for a configuration setting at a hierarchically higher level, such as for this group's parent group or a device group above that,    displays next to the setting's name.
  7. For information on a specific device setting, click for help.
  8. To configure device settings for a different type of SIP IP phone device, use the Settings for drop-down list to change from Pingtel to Cisco or vice versa.
  9. Click Save when your settings are complete.

Changes to a device group's settings do not take effect until you send profiles to the group.

Send profiles

To send new and changed settings to all of the devices in a selected device group, you:

  1. Click List All Device Groups in the left column.
  2. Find the device group.
  3. Select Send Profiles on the Actions list for that group.
  4. Click Go. A pop up window opens.
  5. Check the check box for:
    • Device settings if you have changed any settings for the device group, or changed device assignments to different users.
    • User settings if you have changed any settings for a user who has a device in this group assigned.
    • Application settings if you have changed the applications assigned to a user with a device in this group.

If you are not sure which settings changed, leave all three check boxes checked.

  1. Click Send Profiles.

SIPxchange generates and sends the selected profiles to the phones associated with devices in this group. The phones also restart automatically, and the new settings take effect.

For reference, each time you send profiles the Job Status log records and displays profile status information.

Restart devices

To restart all of the phones associated with devices that have been organized into a device group, you:

  1. Click List All Device Groups in the left column.
  2. Find the device group.
  3. Select Restart Device Group Device(s) on the Actions list for that group.
  4. Click Go.

Other options

Delete a device group

Deleting a device group removes it from the Configuration Server. Unless you assign them to other groups first, you also delete:

  • All devices assigned to that group.

  • All subsidiary device groups.

  • All devices assigned to subsidiary device groups.

Before you delete a group, you may wish to change the group assignments for the devices in that group.

To delete a device group, you:

  1. Click List All Device Groups in the left column.
  2. Find the group you want to delete.

  3. Select Delete Device Group on the Actions list for that group.

  4. Click Go.
