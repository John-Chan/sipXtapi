Add a user

To add a user, you specify identifying information and assign that user to a group.

  1. Click List All Users in the left column.
  2. Click Add User at the top of the Users page. A pop up window opens.
  3. Enter information about the user, including:
    • User ID (which must be unique)
    • PIN (Personal Identification Number)
    • User Group
  4. Optionally, you can identify the user by first and last name. If supplied, these values identify calls made from this user's assigned devices.
  5. Optionally, you can assign an extension to the user. A numeric extension is useful if your user IDs are alphanumeric; if assigned, extensions must be unique.
  6. Click Submit.

The pop up window remains open so that you can:

  • Add another user.
  • Return to the Users page: click Close.

User-specific settings

  1. Click List All Users in the left column.

  2. Click the folder icon next to the group to which the user is assigned, then click the user ID of the user you want to configure.
    Alternatively, click Search and supply all or part of the user ID.
    A new page opens with the user's General settings.

  3. You can assign devices that have different device types to any user. To assure that phones of any given type are correctly configured for this user's needs, indicate the settings you wish to configure first at the top of the page: select Pingtel or Cisco from the Settings for drop-down list.

    • Different categories of settings display based on your selection.
    • Settings for the General and Permission categories apply to the entire group, regardless of device type.

  4. To override values inherited from this user's group, or to set values that apply to this user only, click on a category name at the top of this page. The settings in that category display.
    Use the General category to change the identifying information for a user, or to change a user's group assignment.

  5. You define configuration settings either by selecting one option from a provided list, making changes directly in a text box, or by clicking    to add or    to edit a set of fields in a pop up window.

  6. If a value has been defined for a configuration setting at a hierarchically higher level, such as for this user's parent group or a user group above that,    displays next to the setting's name.

  7. For information on a specific user setting, click    next to the setting's name for help.

  8. To configure user settings that apply to a different type of phone, use the Settings for drop-down list at the top of the page to change from Pingtel to Cisco or vice versa.

  9. Click Save when your settings are complete.

Changes to user-specific settings do not take effect until you send profiles to the user's devices.

Assign devices to a user

Before you assign a device to a user, you must set up at least one device. To assign a device to a user, you:

  1. Click List All Users in the left column.
  2. Click the folder icon next to the group to which the user is assigned.
  3. Find the user to whom you want to assign a device.
  4. Select Assign Devices on the Actions list for that user.
  5. Click Go. A pop up window opens.
  6. Choose a Device from the list of available devices that displays.
  7. Click Assign.

Changes to device assignments do not take effect until you send profiles.

The pop up window remains open so that you can:

  • Assign another device to the same user.
  • Unassign a currently assigned device.

  • Return to the Users page: click Close.

Tip: To assign a device to a different user, leave the pop up window open, then follow steps 3 through 7 to select a different user and assign a different device.

To change the device(s) assigned to a user: follow steps 1 through 4, then either choose another device to assign or an existing device to unassign.

Assign applications to a user

Before you assign an application or application set to a user, you must set up at least one application or set. To assign applications to a user, you:

  1. Click List All Users in the left column.
  2. Click the folder icon next to the group to which the user is assigned.
  3. Find the user to whom you want to assign applications.
  4. Select Assign Applications on the Actions list for that user.
  5. Click Go. A pop up window opens.
  6. To work with individual applications, click the Applications radio button at the top of the box.
    To work with application sets, click the Application Sets radio button.
  7. Choose an Application or Application Set from the list that displays.
  8. Click Assign.

The pop up window remains open so that you can:

  • Assign other applications or application sets to that user.
  • Unassign applications or application sets.

  • Return to the Users page: click Close.

To change the application(s) assigned to a user: follow steps 1 through 4, then choose applications or application sets to assign or unassign.

Changes to application assignments do not take effect until you send profiles.

Tip: Only Pingtel phones run applications.

Send profiles

To send configuration settings, and device and application assignments to all of the devices assigned to a selected user, you:

  1. Click List All Users in the left column.

  2. Click the folder icon next to the group to which the user is assigned.

  3. Find the user for whom you want to generate and send profiles.

  4. Select Send Profiles on the Actions list for that user.

  5. Click Go. A pop up window opens.

  6. Check each check box to send the corresponding profiles:

    • Device settings if you have changed any configuration settings by selecting the user's device(s), or changed the devices assigned to the user.

    • User settings if you have changed any user-specific configuration settings.

    • Application settings if you have changed the applications assigned to the user.

If you are not sure which settings changed, leave all three check boxes checked.

  1. Click Send Profiles.

The pop up window closes, and SIPxchange generates the selected profile(s) and sends them to all of the phones associated with the device(s) assigned to the user. The phones restart when they are idle (Pingtel phones display a warning first), and the new settings take effect.

For reference, the Job Status log records and displays profile status information each time you send profiles.

Restart devices

To restart all of the phones assigned as devices to a given user, you:

  1. Click List All Users in the left column.
  2. Click the folder icon next to the group to which the user is assigned.
  3. Find the user.
  4. Select Restart User Device(s) on the Actions list for that user.
  5. Click Go.
  6. Click Ok in the confirmation message that pops up.

Other options

Delete a user

To delete a user, you:

  1. Click List All Users in the left column.
  2. Click the folder icon next to the group to which the user is assigned.
    Alternatively, click Search and enter all or part of the user ID.
  3. Find the user you want to delete.
  4. Select Delete User on the Actions list for that user.
  5. Click Go.
  6. Click Ok in the confirmation message that pops up.

SIPxchange automatically unassigns any devices assigned to the deleted user so that they can be assigned to another user.
