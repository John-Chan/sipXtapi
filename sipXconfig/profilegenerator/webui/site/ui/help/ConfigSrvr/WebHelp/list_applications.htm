Download an application

To download an application published by Pingtel to your local, SIPxchange Web server:

  1. Click List All Applications in the left column.
  2. Click Download Application at the top of the page.
  3. Review the applications that display.
    Use the drop-down list at the top of the pop up window to review different groups of applications.
  4. Click Download next to each application you wish to store locally.
  5. To return to the List All Applications page, click Close.
    The downloaded applications display.

After download, repeat these steps periodically and click Reload to assure that you have the most recent versions of your selected applications.

Other options

Add an application

To add an application:

  1. Click List All Applications in the left column.
  2. Click Add Application at the top of the List Applications page.
  3. Enter the following information about the application:
    • Name
    • URL
    • Description
  4. Click Submit.

After you add an application, you can:

  • Add another application.
  • Return to the List All Applications page: click Close.

Other options

Change application data

To change the information for an existing application:

  1. Click List All Applications in the left column.
  2. Click on the name of the application you want to change.
  3. Change the application's name, URL, or description.
  4. Click Save.

Other options

Delete an application

Deleting an application removes it from the system, and from any application sets to which it may have belonged.

To delete an application, you:

  1. Click List All Applications in the left column.
  2. Find the application you want to delete.

  3. Click the icon in the Delete column for that application.
