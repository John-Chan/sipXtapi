Welcome to the Pingtel Configuration Server!

The Pingtel Configuration Server™ is a tool for managing a large number of SIP phones in an organization. With the Configuration Server, you identify each device and each user in the system. You set up:

  • Users and User groups

  • Devices and Device groups

  • Applications and Application sets

You organize this data by creating hierarchical groups for devices and users, and by assigning devices and applications to users. You also specify configuration settings for users and devices: settings defined for a group are inherited by all subsidiary groups and the users or devices in those groups.

The Configuration Server then automates delivery of software and data to the SIP phones in your organization.

The Configuration Server also offers administrative options, including options to change the administrative password, report job status, or manage extension pools. You can also configure system-wide services: for help on working with services, click an option under Services then Help.

This online help system provides step-by-step procedures to help you perform tasks with the SIPxchange Configuration Server. Administering SIPxchange provides conceptual information. To review or download a copy of this guide, visit our Web site at www.pingtel.com/docs.

You can also work with the SIPxchange Comm Server: for help on setting Global Definitions or other system services, click an option in the navigation bar at left, then Help.

Tip: For best results, you should use Internet Explorer version 6.00 or above, or Netscape version 6.2 or above, to access SIPxchange. Your browser should not cache pages. For example:

  • For Internet Explorer 6.0, from the Tools menu select Internet Options. On the General tab, under Temporary Internet files click Settings. Under Check for newer versions of stored pages, choose the Every visit to the page option.

  • For Netscape 7.0, from the Edit menu select Preferences, click on Advanced, then Cache, and under Compare the page in the cache to the page on the network, choose the Every time I view the page option.
