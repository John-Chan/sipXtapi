About users

Work with users

The Configuration Server stores a database of the individuals who are the end users of your organization’s SIP phones. On the Configuration Server, users are the subscribers to the SIPxchange system. While a user will often represent an actual person, you can also set up a user to represent a certain role (such as Reception or Security), or the set of features you wish to make available on certain phones (such as Public lobby phone or Loading Dock).

With the Configuration Server, you can assign zero, one, or more devices to each user.
