Job Status

This option displays status information for send profile requests, including the date and time of the request, the name of the user, device, or group and the type of profile requested, and the status.

The Configuration Server delivers data and software to the phones in your system only when you send profiles. You can send profiles for a:

  • Device: the associated phone receives new data and software, then restarts.

  • Device Group: all of the phones associated with devices in the group receive new data and software, then restart.

  • User: the phone associated with the device assigned to the user receive new data and software, then restarts. If you assign more than one device to the user, all of them receive the profile and restart.

  • User Group:  the phones associated with the devices assigned to all users in the group receive new data and software, then restart.
