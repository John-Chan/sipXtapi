Add an application set

To add an application set:

  1. Click List All Application Sets in the left column.
  2. Click Add Application Set at the top of the page. A pop up window opens.
  3. Enter a name to identify the application set.
  4. Click Submit.

After you finish adding an application set, you can:

  • Add another application set.
  • Return to the List Application Sets page: click Close.

Other options

Group applications into a set

Before you can add an application to an application set, you must add that application to the Configuration Server.

To assign an application to an application set, or to change the applications currently assigned to an application set:

  1. Click List All Application Sets in the left column.
  2. Find the application set to which you want to assign applications.
  3. Click the icon in the Assign Apps column for that set. A pop up window opens.
  4. To add an application to the set, select an application from the drop-down list at the top of the window, then click Assign.
  5. To remove one of the applications that is already in the set, select an application from the drop-down list in the center of the window, then click Unassign.
  6. Click Close when you are finished.

Other options

Change the name of an application set

To change the name of an application set:

  1. Click List All Application Sets in the left column.
  2. Click on the name of the application set you want to rename.
  3. Enter a new name for the set.
    You can also review the list of applications assigned to the set on this page.

  4. Click Save.

Other options

Delete an application set

To delete an application set:

  1. Click List All Application Sets in the left column.
  2. Find the application set you want to delete.
  3. Click the icon in the Delete column for the set you want to remove.

The applications previously assigned to the set are not deleted.
