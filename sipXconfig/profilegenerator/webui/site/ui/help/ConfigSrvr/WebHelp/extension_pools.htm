Add an extension pool

To add an extension pool:

  1. Click Extension Pools in the left column.
  2. Click Add Extension Pool at the top of the page. A pop up window opens.
  3. Enter a name to identify the extension pool.
  4. Click Submit.

The pop up window remains open so that you can:

  • Add another extension pool.
  • Return to the Extension Pools page: click Close.

Other options

Assign extensions to a pool

To assign extensions to an extension pool:

  1. Click Extension Pools in the left column.
  2. Find the extension pool to which you want to assign extensions.
  3. Select Add Extensions on the Actions list for that pool.
  4. Click Go. A pop up window opens.
  5. You can assign a single extension number, or a consecutive range of extension numbers to the pool:
    • To assign a single extension number, enter that extension number in both the Min Extension and Max Extension fields.
    • To assign a range of extension numbers, enter the starting extension number in the Min Extension field, and the ending number in the Max Extension field.
  6. Click Submit.
  7. Click Close when you are finished.

Other options

Remove extensions from a pool

To remove extensions from an extension pool:

  1. Click Extension Pools in the left column.
  2. Find the extension pool from which you want to remove extensions.
  3. Select Delete Extensions on the Actions list for that pool.
  4. Click Go. A pop up window opens.
  5. You can remove a single extension number or a range of extension numbers:
    • To remove a single extension number, enter that extension number in both the Min Extension and Max Extension fields.
    • To remove a range of extension numbers, enter the starting extension number in the Min Extension field, and the ending number in the Max Extension field.
  6. Click Submit.
  7. Click Close when you are finished.

Tip: If you remove an extension that is currently assigned to a user from a pool, the extension is also deleted from that user's general information.

Other options

Change the name of an extension pool

To change the name of an extension pool:

  1. Click Extension Pools in the left column.
  2. Click on the name of the extension pool you want to rename.
  3. Enter a new name for the pool.
    You can also review the different extension numbers assigned to the pool on this page.

  4. Click Save.

Tip: You cannot change the name of the reserved extension pool.

Other options

Delete an extension pool

If you delete an extension pool and some or all of its extensions are currently assigned to users, those extensions will also be deleted from the users' general information. You may wish to change the user's data first.

To delete an extension pool:

  1. Click Extension Pools in the left column.
  2. Find the extension pool you want to delete.
  3. Select Delete Extension Pool on the Actions list for that device.
  4. Click Go.

Tip: You cannot delete the reserved extension pool.
