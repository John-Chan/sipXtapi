List All Administrators

You manage the administrative user "superadmin" for the Configuration Server from this page.

Change password

To change the password for the superadmin administrator:

  1. Click List All Administrators in the left column.
  2. Click the icon in the Change Password column. A pop up window opens.
  3. Enter the new password.
  4. Confirm the new password.
  5. Click Submit.
