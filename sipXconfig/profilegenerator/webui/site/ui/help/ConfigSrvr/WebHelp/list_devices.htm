Add a device

  1. Click List All Devices in the left column.
  2. Click Add Device at the top of the Devices page. A pop up window opens.
  3. Enter information about the device, including:
    • Device Name
    • Device Type
    • Serial Number
      • For Pingtel phones, the first 6 characters are supplied based on the selected device type. You supply the remaining 6 characters, which must be digits or the letters A to F in upper or lower case.
      • For Cisco phones, you supply the 12 character MAC address.
    • Description
    • Device Group
  4. Click Submit.

The pop up window remains open so that you can:

  • Add another device.
  • Return to the Devices page: click Close.

Configure device-specific settings

  1. Click List All Devices in the left column.
  2. Click the folder icon next to the group to which the device is assigned, then click the name of the device you want to configure.
    Alternatively, click Search and enter all or part of the Device Name.
    A new page opens with the device's General settings.
  3. To override values inherited from this device's group, or to set values that apply to this device only, click on a category name at the top of this page. The settings in that category display.
    • Use the General category to change the identifying information for the device, or to change its group assignment.
    • Other categories display settings appropriate for this device's device type.

  4. You define configuration settings by selecting one option from a provided list, making changes directly in a text box, or by clicking    to add or    to edit a set of fields in a pop up window.

  5. If a value has been defined for a configuration setting at a hierarchically higher level, for this device's device group or a device group above that,    displays next to the setting's name.

  6. For information on a specific device setting, click    next to the setting's name for help.

  7. Click Save when your settings are complete.

Changes to device-specific settings do not take effect until you send profiles to the device.

Tip: You cannot change the device type of an existing device. To replace a Cisco 7940 phone with a Pingtel xpressa phone, for example, you must add a new device to represent the Pingtel phone, then delete the existing device that corresponds to the Cisco phone.

Send profiles

To generate profiles of new and changed settings and send them to a selected device, you:

  1. Click List All Devices in the left column.
  2. Click the folder icon next to the group to which the device is assigned.
    Alternatively, click Search and enter all or part of the Device Name.
  3. Find the device.
  4. Select Send Profiles on the Actions list for that device.
  5. Click Go. A pop up window opens.
  6. Check the check box for:
    • Device settings if you have changed any settings for the device, or changed the assignment of the device to a different user.
    • User settings if you have changed any settings for the user associated with this device.
    • Application settings if you have changed the applications assigned to the user of this device.

If you are not sure which settings changed, leave all three check boxes checked.

  1. Click Send Profiles.

The phone associated with the device restarts, and the new settings take effect.

For reference, each time you send profiles the Job Status log records and displays profile status information.

Restart device

To restart the phone associated with a device, you:

  1. Click List All Devices in the left column.
  2. Click the folder icon next to the group to which the device is assigned.
    Alternatively, click Search and enter all or part of the Device Name.
  3. Find the device.
  4. Select Restart Device on the Actions list for that device.
  5. Click Go.

Delete a device

Reminder: To replace a phone with another of the same type, you can change the data for the device that represented the old phone by updating its serial number to match a new phone. Delete a device only when there is no longer any need to maintain its distinct characteristics in your organization.

To delete a device, you:

  1. Click List All Devices in the left column.
  2. Click the folder icon next to the group to which the device is assigned.
    Alternatively, click Search and enter all or part of the Device Name.
  3. Find the device you want to delete.

  4. Select Delete Device on the Actions list for that device.

  5. Click Go.

Deleting a device removes it from the system.

Tip: To restore a phone's configuration settings to factory defaults, and remove any potentially sensitive data, on the phone top:

  1. Press MORE.

  2. On the Apps tab, press Prefs.

  3. Press MORE.

  4. On the Menu tab, press Factory Defaults.
