When set to Do Not Disturb

With this setting, you specify how calls will be handled when Do Not Disturb is set to ENABLE.

  • To send a busy response to all SIP call invitations, set this parameter to SEND_BUSY. For users with the voicemail permission, calls are forwarded to the voicemail inbox.

  • To use the user's call forwarding preferences:

    • choose FORWARD_ON_NO_ANSWER to send incoming calls to the Forward on No Answer to destination

    • choose FORWARD_ON_BUSY to send calls to the Forward if Busy to destination.

For users with the voicemail permission, calls are forwarded to these destinations if they are defined; otherwise, calls are forwarded to the voicemail inbox.

Parameter name

PHONESET_DND_METHOD

Type

Enumeration

Default if blank

FORWARD_ON_BUSY

Values

FORWARD_ON_BUSY, FORWARD_ON_NO_ANSWER, SEND_BUSY

User Category

Call Handling
