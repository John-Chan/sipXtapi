Identify outgoing calls with this SIP URL

When an end user makes a call from a phone, all SIP messages sent by that phone include a single selected SIP URL, which identifies the source of the call. Caller ID services, for example, make use of this information. This setting identifies which SIP URL to send.

From among all of the lines associated with a given Pingtel phone, including both user lines and the device line, you specify the SIP URL of one line as the identity to send with outgoing calls.

Parameter name

USER_DEFAULT_OUTBOUND_LINE

Type

String

Default if blank

PHONESET_LINE

Values

For one line, select the complete SIP URL and copy it. Paste the value into the text box for this setting.

User Category

Lines
