Incoming call alert

This setting defines the alert method to use for incoming calls.

  • VISUAL flashes the Pingtel phone’s message waiting indicator.

  • AUDIBLE plays the ring tone. (If a call is already in progress, a call waiting beep plays if the If a call is in progress, play call waiting alert or act as busy setting is ALERT.)

  • BOTH results in both alert types.

Parameter name

PHONESET_RINGER

Type

Enumeration

Default if blank

BOTH

Values

AUDIBLE, BOTH, VISUAL

User Category

Call Handling
