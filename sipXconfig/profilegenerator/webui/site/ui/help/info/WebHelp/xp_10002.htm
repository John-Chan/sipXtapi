Show call handling on phone top

This setting determines whether end users will have access to call handling settings through the Prefs application on the phone top. In a SIPxchange system, all call handling settings must be set using the browser-based SIPxchange interface for end users, or by a system administrator using the SIPxchange interface for administrators. Any changes made to call handling settings on the phone top are temporary.

As a result, this parameter should be set to DISABLE for all devices.

Parameter name

PHONESET_PREFS_CALL_HANDLING

Type

Enumeration

Default

ENABLE

Values

DISABLE, ENABLE

Device Category

Phone Top

Reminder: Supply DISABLE for this setting for the top-most group in your device group hierarchy. For devices with this setting enabled, end users may believe that they can change their call handling settings on the phone top rather than through the SIPxchange interface.
