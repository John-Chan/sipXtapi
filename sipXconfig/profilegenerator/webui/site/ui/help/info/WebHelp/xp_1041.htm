Codec priority list

When a Pingtel phone engages in call setup with another SIP user agent, they negotiate to determine which audio codec to use for the call. This setting lists all of the codecs supported by Pingtel phones. The order in which audio codecs appear in this list indicates the preference to use when the phone associated with this device negotiates with other SIP user agents. Codecs that appear earlier in the list are preferred over those that appear later in the list.

Pingtel phones support one DTMF codec and up to five audio codecs:

You enter all codec IDs separated by spaces. Since the telephone-event codec ID corresponds to a DTMF codec (and not an audio codec), its use is not affected by its position in the list relative to other codecs.
