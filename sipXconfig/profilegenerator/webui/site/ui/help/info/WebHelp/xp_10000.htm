Permissions

Settings in this category control whether or not the selected user, or group of users, is authorized to use certain phone system features. The default (if blank) for all permission settings is DISABLE.

User can dial 900 numbers

  • To allow calls to 900 numbers, select ENABLE.

  • To prevent 900 calls, select DISABLE.

List user in Auto Attendant

  • To include this user or group of users in the Auto Attendant directory, select ENABLE.

  • To prevent this user or group from being included, or to remove this user or group from the directory, select DISABLE.

User can dial international numbers

  • To allow international calls, select ENABLE.

  • To prevent international calls, select DISABLE.

User can dial local numbers

  • To allow local calls, select ENABLE.

  • To prevent local calls, select DISABLE.

User can dial long distance numbers

  • To allow long distance calls, select ENABLE.

  • To prevent long distance calls, select DISABLE.

User has voicemail inbox

  • To set up a voicemail inbox for this user, or all users in this group, select ENABLE.

  • To leave this user or group without SIPxchange voicemail,  or to remove an existing inbox or set of inboxes, select DISABLE.

User can forward calls to external numbers

  • To allow this user, or all users in this group, to specify external, PSTN addresses for forwarding calls, select ENABLE.

  • To prevent this user, or all users in this group, from forwarding calls to external, PSTN addresses, select DISABLE.

User can record system prompts

  • To enhance the voicemail interface that this user, or all users in this group,  accesses from a phone top with options to record system-wide greetings and prompts, select ENABLE.

  • To provide this user or group with the standard voicemail interface, select DISABLE.
