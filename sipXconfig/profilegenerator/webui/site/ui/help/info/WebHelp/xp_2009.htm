Retrieve voicemail messages from

This setting defines the retrieval address for voicemail messages on a network voicemail server. By defining the address of a target voicemail inbox, you enable one-step retrieval of messages in the user's voicemail inbox from the phone top.

Parameter name

PHONESET_VOICEMAIL_RETRIEVE

Type

String

Examples

10

sip:sub-username-retrieve-inpin@sipserver
(for SIPxchange systems that do not include the Media Server)
 

Note

If you do not use the SIPxchange Media Server  voicemail service, refer to the documentation supplied with your voicemail server to determine the appropriate syntax for identifying this SIP URL .

User Category

Network

Reminder: To set up SIPxchange voicemail for a user or group of users, you enable the User has voicemail inbox permission.
