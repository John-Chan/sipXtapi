Speed dial numbers

When you set up a speed dial number, you store four different values:

  • The Number that you want to dial instead of the full phone number or SIP URL

  • An identifying Name

  • The Type of value that will actually be dialed: either a phone number or a SIP URL

  • The value to dial: a phone Number or SIP URL
    For phone numbers, these entries are valid:

    • dial pad characters: 0 to 9 * and #

    • separator characters: spaces and - + .

    • special characters p to indicate a pause and i to indicate an interactive wait; for example, when used to dial an IVR (Interactive Voice Response) system

For SIP URLs, all keyboard characters are vaild.

To add a speed dial number, click the add icon    and enter your values in the popup window that opens. When you click Submit, these values are stored together in a set of four properties.

To edit or delete a speed dial number, click the edit icon    or the delete icon  .

Parameter names

PHONESET_SPEEDDIAL.<n>.ID
PHONESET_SPEEDDIAL.<n>.LABEL
PHONESET_SPEEDDIAL.<n>.TYPE
PHONESET_SPEEDDIAL.<n>.ADDRESS

Type

ID, LABEL, and ADDRESS store strings; TYPE is an enumeration

Values

for TYPE: choose NUMBER or URL

Examples

To dial just 1 to reach your home phone number at
1-518-875-8060:
Speed dial number: 1
Name: home
Type: NUMBER
Number/URL: 15188758060

To dial just 4 to reach your company's help desk at
sip:help@yourco.com:
Speed dial number: 4
Name: help desk
Type: URL
Number/URL: sip:help@yourco.com

User Category

Speed Dial

Reminder: Individual end users can set up and maintain their own speed dial numbers by logging in to the Configuration Server with their assigned user IDs and PINs.
