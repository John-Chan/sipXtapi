Encrypt Phone Profiles

Applies to instant xpressa softphones only.

With this setting, you control encryption of configuration data to protect instant xpressa softphones from unauthorized configuration changes.

  • Protects the softphone's pinger-config, user-config, and app-config files (stored in the local Pingtel\instantxpressa\env directory) from manual changes.

  • Protects the configuration profiles stored on the SIPxchange server and delivered over the network from unauthorized retrieval.

  • Requires manual, phone top entry of a valid SIPxchange user name and PIN each time new configuration profiles are sent to the phone.

Note: If you change this setting to ENABLE, you must also change the PIN (Personal Identification Number) for the user to whom you have assigned this device. If you change this setting for a device group, the PINs for all users with device assignments from that group must be changed.

Parameter name

ENCRYPT_PROFILES

Type

Enumeration

Default

DISABLE

Values

ENABLE, DISABLE

Device category

Security
