Device Locale

This setting identifies the country where this phone is located by storing a two-character ISO country code.  Using the specified country code, the phone associated with this device plays locale-specific versions of these call progress tones:

  • Dial Tone: the phone system is ready to receive call information

  • Ringback Tone: a connection has been made and the dialed address is ringing

  • Busy Tone: the destination address is busy

  • Call Failed Tone: the destination address cannot be reached for a reason other than busy

Parameter name

PHONESET_LOCALE_COUNTRY

Type

String

Default

US

Values

DE (Germany)

FR (France)

GB (Great Britain)

IL (Israel)

NL (Netherlands)

US (United States)

If you supply a code that is not on this list, the phone uses US
