Title Bar Content

This setting customizes the information that displays in the title bar of a Pingtel phone's home screen. By default, phones display the time and date, then, after 5 seconds, change to display the phone's SIP URL. This default is expressed as:

{h:mm a MM/dd/yy} \n {displayname}<{sipurl}>

You can specify:

  • a text string

  • one or more of the options for phone identification that follow. These options use data from the SIP URL that is designated as the outbound calling line (see Identify outgoing calls with this SIP URL).

Option

Description

{displayname}

The display name from the SIP URL.

{hostname}

The domain name of the host in the SIP URL.

{sipurl}

The phone's SIP URL in the format {resource ID}@{host}.

{username}

The resource ID from the phone's SIP URL.

\n

Defines a complete display segment. Each segment displays for 5 seconds, then the next segment displays. There is no limit on the number of segments you can define.
