General information

This category displays identifying information for the selected device. You can change the device's:

  • Name

  • Serial Number

  • Device Group

  • Description

You can also select another category to review or change settings.

Click Save when you complete all of your changes.

Tip: You cannot change the device type for an existing device. To replace a Cisco 7940 phone with a Pingtel xpressa phone, for example, you must set up a new device to represent the Pingtel phone, then delete the existing device that corresponds to the Cisco phone.
