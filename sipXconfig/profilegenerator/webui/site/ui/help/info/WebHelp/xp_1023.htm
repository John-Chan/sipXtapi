Send registration messages every <n> seconds

This setting defines the number of seconds until the registration of all lines for the Pingtel phone associated with this device expires. The phone automatically re-registers all lines with their associated registry servers, for example, the SIPxchange SipRegistrar, before this time period elapses, or before the registry server's defined register period elapses, if it is shorter.

Parameter name

SIP_REGISTER_PERIOD

Type

Integer

Default if blank

3600

Notes

This time period applies to all lines configured to register with a host.

Device Category

SIP servers

Reminder: Your SIPxchange SipRegistrar is the host in the SIP URLs of the device line and the primary user line. You can specify the SipRegistrar or another registry server as the host in additional user lines, and define whether or not each additional line should register.
