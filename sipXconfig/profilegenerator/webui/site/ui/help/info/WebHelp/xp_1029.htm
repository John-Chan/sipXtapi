Send Java debugging output to

With this setting, you control the destination of Java debugging output produced by the phone associated with this device.

  • To produce Java debugging output, set this value to CONSOLE. You must also set Activate console logging to ENABLE.

  • To prevent Java debugging output from being produced, set this value to <BLANK>.

Parameter name

JAVA_OUTPUTSTREAM_REDIRECT

Type

Enumeration

Default

null (disabled)

Values

CONSOLE

Device Category

Diagnostics

Reminder: If you choose CONSOLE, you must also set Activate console logging to ENABLE.
