Activate Telnet access

With this setting, you control Telnet access to this device.

  • When set to DISABLE, Telnet access is not permitted.

  • Under the direction of Pingtel's Technical Assistance Center (TAC) only, you can set this value to ENABLE to allow Telnet sessions.

The Telnet session requires the admin user name and password. A password must be defined for the admin user name (it cannot be left null). See Authorized phone users.

Parameter name

PHONESET_TELNET_ACCESS

Type

Enumeration

Default if blank

DISABLE

Values

DISABLE, ENABLE

Device Category

Security
