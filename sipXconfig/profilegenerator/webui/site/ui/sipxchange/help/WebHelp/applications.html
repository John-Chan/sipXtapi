Manage Applications

On this page, you work with applications for your Pingtel phone.

  • Install an application

  • Remove an application

To review the applications that you have installed on your Pingtel phones, see the list of Currently Loaded Applications at the bottom of the page.

Tip: This list does not include any applications that may have been installed for you by your system administrator. For a complete list of applications, press MORE then Apps on your Pingtel phone.

Install an application

To install an application on your Pingtel phone:

  • Select the application you want to install from the list of Applications.

  • Click Install.

When your work with applications is complete, click Send Application Profile at the bottom of the page. This updates the applications installed on all of your Pingtel phones after a restart.

Tip: If you use an instant xpressa softphone, you may need to enter your PIN on the phone top for new applications to become active.Your system administrator controls this security setting.

Remove an application

To remove an application:

  • Select the application from the list of Existing Applications.

  • Click Uninstall.

When your work with applications is complete, click Send Application Profile at the bottom of the page. This updates the applications installed on all of your Pingtel phones after a restart.

Tip: If you use an instant xpressa softphone, you may need to enter your PIN on the phone top for changes to your applications to become active.Your system administrator controls this security setting.
