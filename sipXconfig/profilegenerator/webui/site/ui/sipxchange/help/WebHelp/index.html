Welcome

Pingtel's SIPxchange gives you a desktop interface for working with your phone. You can:

  • Manage Speed Dial Numbers

  • Manage Call Handling Settings

  • Manage Applications

  • Change your PIN

These features apply to the Pingtel phones you use with the SIPxchange phone system. If you use a Cisco phone, only the Change PIN option displays.

You can also work with voicemail (if you have a voicemail inbox): for help on working with voicemail, click an option under either Voicemail Folders or Voicemail Preferences, then Help.

 

Tip: For best results, use Internet Explorer version 6.00 or above, or Netscape version 6.2 or above, to access the SIPxchange interface. Your browser should not cache pages. For example:

  • For Internet Explorer 6.0, from the Tools menu select Internet Options. On the Advanced tab, under Temporary Internet files click Settings. Under the Check for newer versions of stored pages option, choose the Every visit to the page option.

  • For Netscape 7.0, from the Edit menu select Preferences, click on Advanced, then Cache, and under Compare the page in the cache to the page on the network, choose the Every time I view the page option.

Please contact your System Administrator for further assistance with the SIPxchange system.
