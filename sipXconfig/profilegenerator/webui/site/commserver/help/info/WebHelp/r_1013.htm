IP address and port for Voicemail requests

Identifies the host and port that the SipRegistrar uses to send HTTP or HTTPS requests to the Media Server’s Voicemail application.

Property

SIP_REGISTRAR_VOICEMAIL_SERVER

Type

String

Default

Uses HTTPS and the local IP address at port 8091:

https%3A%2F%2Flocalhost%3A8091
