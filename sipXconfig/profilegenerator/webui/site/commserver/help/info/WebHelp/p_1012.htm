Enable SipAuthProxy component

Enables and disables use of the SipAuthProxy. When set to ENABLE:

  • The SipProxy routes all requests that do not match rules in the Proxy Forwarding data set to the SipAuthProxy

  • You should also set the host address for the SipAuthProxy with the Address of SipAuthProxy component setting

Property

SIP_PROXY_USE_AUTH_SERVER

Type

Enumeration

Default

ENABLE

Values

ENABLE or DISABLE
