Welcome to the Pingtel Comm Server!

With the Pingtel Comm Server™, you configure settings for system-wide services, including:

  • Global Definitions

  • Gateways

  • Dial Plans

  • SipAuthProxy

  • SipProxy

  • SipRegistrar

  • SipStatus

You can also control the operation of the SIPxchange's component services.

You also use the SIPxchange interface to organize data and automate software delivery to the SIP phones in your ordanization. For help on working with users, devices, or applications, click another menu option then Help.

This online help system provides step-by-step procedures to help you perform tasks for the SIPxchange Comm Server. Administering SIPxchange provides conceptual information about SIPxchange. To review or download a copy of this guide, visit our Web site at www.pingtel.com/docs.

You can also work with the SIPxchange Configuration Server: for help on working with users, devices, or other features, click an option in the navigation bar at left, then Help.

Tip: For best results, you should use Internet Explorer version 6.00 or above, or Netscape version 6.2 or above, to access SIPxchange. Your browser should not cache pages. For example:

  • For Internet Explorer 6.0, from the Tools menu select Internet Options. On the Advanced tab, under Temporary Internet files click Settings. Under the Check for newer versions of stored pages option, choose the Every visit to the page option.

  • For Netscape 7.0, from the Edit menu select Preferences, click on Advanced, then Cache, and under Compare the page in the cache to the page on the network, choose the Every time I view the page option.
