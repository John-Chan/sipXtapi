Manage Dial Plans

You define dial plans to establish how your SIPxchange IP voice system will route incoming and outgoing calls. You configure the system-wide dial plans on this page.

Dial plan info

Internal station numbering format  

You identify Internal station numbers, or extensions, by defining their length and format. The dial plan for an internal number consists of:

  • one or more digits, optionally restricted to a possible range
    To specify a digit in a range, supply [digit-digit]

  • one or more wildcard digits
    To specify a wildcard, supply x.

For example, if your dial plan is [4-6]xxx, then a dialed number that begins with 4, 5, or 6, and that is four digits in length,  will match.

The default format is [1-7]xx.

Prefix for Direct Inward Dialing (DID)  

If your SIPxchange IP voice system uses DID numbers, you set this prefix to the constant digits that get combined with an internal extension to produce a full DID number.

For example, if your DID number prefix is 7818543, you supply 7818543. If the format of your internal station numbers is [1-7]xx, then extension 678 will correspond to DID number 781 854 3678.

By default, no value is set.

Reminder: You only define a DID prefix if you use DID numbering.

To call the Auto Attendant, dial  

Identifies the system extension reserved for the Auto Attendant. The default is 100.

To retrieve Voicemail, dial  

Identifies the system extension reserved for Voicemail. The default is 101.

Prefix for dialing PSTN calls  

You identify outbound calls that must be routed through an IP-to-PSTN gateway by establishing a dial plan prefix. The default for dialing PSTN calls is 9.

Prefix for dialing an extension's Voicemail inbox  

To route calls directly to an extension's Voicemail inbox, you supply a prefix. The default is 8.

Prefix for dialing international calls

You identify international calls by establishing a dial plan prefix. The default for dialing international calls is 011.

Configure dial plans

Tip: If your SIPxchange installation has complicated call routing needs, you may not be able to use this procedure to configure your dial plans. Please contact Pingtel’s TAC for more information.

To configure system-wide dial plans:

  1. Click Configuration in the left column.
  2. Find Dial Plans on the list of services.
  3. Click the icon in the Configure column for Dial Plans. A new page lists current values.
  4. You define dial plans by making changes directly in the text boxes.
  5. Click Save when you complete your edits.

Your new settings do not take effect until you restart.

Tip: To review suggestions for phone digit maps that are based on your current system dial plans, click View Digit Maps at the top of the page.
