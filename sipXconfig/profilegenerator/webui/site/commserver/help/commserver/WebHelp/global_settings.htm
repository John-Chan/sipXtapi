Global Settings

When you install the Pingtel Configuration Server, you define the types of phones that you will maintain, and the core software (operating system and firmware) running on those phones. These definitions are global: they apply to all entities managed by the Configuration Server.

Phone Types

You identify different SIP phones by establishing phone types. The distinguishing factors that administrators may choose to define a phone type include manufacturer, vendor, and hardware model.

Currently, the Configuration Server supports SIP phones manufactured by Pingtel Corp. or Cisco Systems, Inc.
