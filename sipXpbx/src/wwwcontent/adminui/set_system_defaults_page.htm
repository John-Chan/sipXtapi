Set System Defaults

You establish system-wide settings for voicemail inbox owners on this page.

You can define:

  • Proxy and registry servers

  • Realm and authentication type

  • inbox owner settings such as message and PIN length

  • Default folders and greetings

  • Prompt settings

  • Housekeeping rules for saving and deleting messages

Set System Defaults

To set system defaults:

  1. Click the Set System Defaults link in the left column.

  2. Clear the values for the options you want to set.

  3. Enter the new values for the options you want to set.

  4. Click Save.

Edit System Defaults

To edit system defaults:

  1. Click the Set System Defaults link in the left column.

  2. Clear the values for the options you want to set.

  3. Enter the new values for the options you want to set.

  4. Click Save.
