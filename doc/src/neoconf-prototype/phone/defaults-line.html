Time Format
(Default: 0)
If 1, time is display using hour range 0-24, otherwise 1-12 AM/PM.

Date Format
(Default: D,Md)
Controls format of date string. D = day of week d = day M = month Up to two comma's may be included. e.g. D,dM = Thursday, 3 July Md,D = July 3, Thursday The field may contain 0, 1 or 2 comma's which can occur only between characters and only one at a time i.e. 'D,,dM' is illegal.
(Default: 1)
If checked, display the day and month in long format (Friday/November), otherwise, use abbreviations (Fri/Nov).
(Default: 1)
If checked, display date above time else display time above date.
