Notable limitations in version 2.8 Release to be addressed in version 3.0

  1. This interface only handles polycom phones. Cisco and sipX phones do not show up here. This interface parallels existing Device and Device Groups pages.
  2. If additional Cicso and sipX phones are assigned to these users you will not see them listed in this interface.
  3. No concept of groups. There will be one place to set default device information until groups can be worked in.
  4. Users and lines will still be configured from old interface. You can still assign cisco and sipx phones to users as well as assign then to polycom phones just not in the same interface.

NOTES:

  1. "Select all pages" will have some user interface representation after clicked to distinguish it from "Select all"
  2. Number of items shown can increase to 20 or so, and should be configurable somewhere.
