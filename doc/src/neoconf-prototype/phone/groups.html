The order of groups is only important when the groups have diffenent values for the same setting. The setting value in the last group has higher precedence.

Groups

Groups allow you organize phones into logical groups and share settings between phones in same group. Groups cannot contain other groups, however phones can be in any number of groups which provides for a much more flexible system.
