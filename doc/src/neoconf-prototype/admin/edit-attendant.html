Action Descriptions
Invalid entry prompt
User will be prompted that, that key is an invalid entry and they will be asked to press another entry
Extension Shortcut ignored.
Dial by name prompt
User will be prompted to dial a person's name using the key pad
Extension Shortcut ignored.
Listen for extension
Attendant will continue to listen for additional key presses until it matches an extension in the internal dialing rules. For example, use must key "1" to listen for 1XX extensions.
Extension Shortcut ignored.
Voicemail login
User will be directed to voicemail system and prompted to log in.
Extension Shortcut ignored.
Extension shortcut
Route call to a special extension. Examples include routing to another auto attendant for chaining menus, a user's extension, directly to user's voicemail
