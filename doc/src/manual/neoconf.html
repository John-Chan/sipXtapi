Neoconf

Neoconf is the codename for a new module created to change the system's architecture over time. It uses more modern J2EE technologies, namely the Spring Framework in an interesting way to stage the evolution across various 6 month releases.

Neoconf is not enbabled by default but it will soon be turned on by default. Here's the procedure for turninng it on

  1. Download Spring Framework (tested with version 1.1.2) Download Url
  2. Unzip Spring Framework anywhere on your system and set environment variable where you've downloaded it 
      export SPRING_HOME=/opt/springframework-1.1.2
  3. Turn on compiling when configuring 
      ./configure --enable-neoconf
  4. Before starting your system, set this environment variable. 
        export PROFILEGENERATOR_OPTS=-Dsipxconfig.facade=org.sipfoundry.sipxconfig.core.SipxConfigFacadeImpl
