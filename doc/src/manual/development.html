Sourcecode Layout JavaDocs

  • common (API) - Common code between all configuration modules
  • profilewriter (API) - Profile Writer Java Source
  • profilepublisher (API) - Profile Publisher Java Source
  • profilegenerator - Profile Generator is broken into a few components
    • ejb (API) - EJB Java Source most of the business logic of the configuration system
    • commserver (API) - Java source to edit configuration files for the sipXproxy, sipXregistry and sipXpublisher
    • authentication (API) - EJB plugin for JBoss handle user authention into the website and SOAP
    • soap (API) - Java source for WAR service that allows SOAP based acess to user line, device, and credential adding, deleting and editing
    • webui (API) - JSP Pages, custom tags, web page logic for web based admin and web user admin WAR file.
    • deploy - Assembles pieces of Profile Generator to create one EAR file that gets installed into JBoss.
